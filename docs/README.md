# Functional Specification

### baseBall

#### Number

- [x] 1-9 사이의 랜덤숫자 생성
- [ ] 3자리 숫자중 맞춘 자리가 있만큼 스트라이크
- [ ] 3자리 숫자중 맞지만 위치가 다른만큼 볼
- [ ] 3자리 숫자중 같은 수 전혀 없으면 낫싱
- [ ] 위 과정 반복후 맞추면 종료
- [ ] 종료후 1번으로 재시작
- [ ] 종료후 2번으로 완전종료
- [x] 잘못된 값 입력시 `throw` 로 예외 발생후 종료처리

- [x] 숫자 입력시 3자리 숫자가 중복되지 않게 입력하기
- [ ] 숫자 입력시 3자리 숫자를 넘어가지 않게 입력하기
